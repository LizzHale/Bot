The Project

Who is Bob?

Wilhelm "Bobnut" DeKokosnoot (aka Bob) is a chat bot that uses natural language processing and machine learning to respond to the contextual polarity of the messages in a chatroom. Bob appears as any other user to chatroom participants. In order to do this, a client socket had to be built to send and receive messages using the Internet Chat Relay (IRC) protocol. All messages received through the socket are sent to Bob who sorts them and sends the appropriate output back.

Programmed in Python, Bob uses a machine learning classifier to categorize the messages received in the chatroom as either positive or negative in sentiment. It is currently trained on a corpus of movie reviews provided by Cornell University. The first classifier built was the naive Bayes which produced a 60% accuracy rate. Bob is now running on a Fisher classifier with 65% accuracy. For a more detailed description of the differences between the two classifiers, please see this page.

The Website

The website you are on is running on Flask, deployed on Heroku, and is monitored by New Relic analytics. Its clean, modern look was designedusing the Kube CSS framework as well as custom elements.

Chat with Bob

To chat with Bob, visit the chat page. The chat client you will be using is powered by KiwiIRC. Choose an available nickname and click "start". Bob will be there to greet you and you can begin chatting with him. As you send Bob messages, he will respond to their sentiment. Bob loves a good knock knock joke. Be sure to tell him one or two while chatting.
